# User Stories - Smart Expense Categorizer

1. **Input Transactions**: As a user, I want to paste my bank transaction text into a text area so that I can quickly process my data.
2. **AI Tagging**: As a user, I want the AI to automatically assign a category (e.g., Shopping, Bills) to each transaction.
3. **Manual Correction**: As a user, I want to be able to change a category if the AI makes an incorrect prediction.
4. **Expense Summary**: As a user, I want to see a total sum of expenses for each category to understand my spending habits.
5. **Delete Entry**: As a user, I want to remove specific transactions from the list before finalizing the summary.
6. **CSV Export**: As a user, I want to download my categorized expenses as a CSV file for use in Excel or Google Sheets.
7. **Mobile View**: As a user, I want to access the app on my phone so I can check my budget while on the go.
8. **Clear All**: As a user, I want a button to clear all current data so I can start a new categorization session fresh.
