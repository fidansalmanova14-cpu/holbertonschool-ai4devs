## Problem Statement
Users often find it tedious to manually categorize their expenses into budgets (e.g., "Starbucks" -> "Food & Drinks"). This leads to poor financial tracking. AI can solve this by instantly identifying the category from raw transaction text.

## MVP Features
- **Transaction Input**: User can paste raw transaction text or upload a CSV.
- **AI Categorization**: Automatically tags each entry with a category (Food, Rent, Tech, etc.).
- **Spending Summary**: A simple visual breakdown (Pie chart/List) of total spending per category.
- **Manual Override**: Ability for the user to manually correct a category if the AI is wrong.
