# User Stories - EcoTrack

## User Story 1: Logging Activities
As a user, I want to log my daily transportation (car, bus, bike) so that I can see its impact on my carbon footprint.

## User Story 2: Real-time Calculation
As a user, I want to see the CO2 estimation immediately after logging an activity to understand the direct impact.

## User Story 3: Daily Eco-Tips
As a user, I want to receive a daily AI-generated tip so that I can learn small ways to improve my sustainability.

## User Story 4: Progress Visualization
As a user, I want to see a chart of my weekly emissions to track if I am improving over time.

## User Story 5: Data Management
As a user, I want to be able to delete a logged activity in case I made a mistake.

## User Story 6: Exporting Results
As a user, I want to download my activity history as a CSV file for further analysis.

## User Story 7: Category Filtering
As a user, I want to filter my history by category (Food, Travel, Energy) to see which area contributes most to my footprint.

## User Story 8: Responsive Access
As a user, I want to access the app on my mobile phone so that I can log activities on the go.
